# take-courses
配合Microsoft自带的Power Automate(以下简述PAD)自动刷课过题(edge浏览器 知到网页端)，适合电脑空闲时间快速刷网课

<img width="415" height="329" alt="image" src="https://github.com/user-attachments/assets/4bf1810d-c71a-4a83-be10-433393175438" />

启动应用，登录Microsoft账号之后edge会自动加载一个插件

<img width="298" height="55" alt="image" src="https://github.com/user-attachments/assets/c09aada2-450d-46ee-b3f4-87b1028fdfd2" />

在PAD中新建流

新建子流程之后将6个子流程内容粘贴(Main,正常播放,初始化播放参数,播放视频,错误处理,小测-跳过)

保存流之后运行，等待流状态显示正在运行即可

<img width="325" height="150" alt="image" src="https://github.com/user-attachments/assets/05046faa-b6a7-46b0-a744-418bd11e230a" />

运行之后点击预先打开的知到课程页面(视频播放界面)，出现下图弹窗即完成了
